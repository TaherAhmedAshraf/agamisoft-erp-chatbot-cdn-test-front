üìã Quick Setup Guide

1Include Required Libraries

Add Socket.IO and Axios to your HTML page:

<script src="https://cdn.socket.io/4.7.4/socket.io.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/axios@1.6.0/dist/axios.min.js"></script>

2Include Chat Widget Files

Add the chat widget CSS and JavaScript files:

<link rel="stylesheet" href="https://your-cdn.com/chat-widget.css"> <script src="https://your-cdn.com/chat-widget.js"></script>

3Initialize the Widget

Add this script to initialize the chat widget:

<script> ChatWidget.init({ serverUrl: 'http://your-backend-url', companyName: 'Your Company', welcomeMessage: 'Hello! How can we help you today?', position: 'bottom-right', theme: 'default' }); </script>
That's it! Your chat widget is now ready. Replace 'http://your-backend-url' with your actual server URL.
